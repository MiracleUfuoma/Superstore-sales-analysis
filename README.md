# Superstore Sales Dashboard

![Power BI Logo](https://upload.wikimedia.org/wikipedia/commons/c/cf/Power_BI_Logo.svg)

## Overview
This Power BI project analyzes the sales performance, profit contribution, and customer retention for a Superstore business. The dashboards provide a comprehensive view of sales trends, profitability by category and region, and insights into top-performing customers and cities.

The report is divided into three main dashboards:
1. **Profit Performance Dashboard**
2. **Sales Performance Dashboard**
3. **Customer Retention Dashboard**


## Key Metrics
- **Total Sales:** $1.6M  
- **Total Profit:** $175.3K  
- **Total Quantity Sold:** 22K  
- **Total Customers:** 773  

These KPIs give a quick snapshot of overall performance.



## Dashboards

### 1. Profit Performance Dashboard
- Visualizes profit by **category** and **region**.  
- Highlights the most profitable categories and regions.  
- Includes a list of cities generating the highest profits.  


### 2. Sales Performance Dashboard
- Shows total sales by **category** and **region**.  
- Highlights top-selling cities.  
- Helps identify which product categories are driving revenue.  
- Provides an intuitive comparison between regions and categories.

### 3. Customer Retention Dashboard
- Lists **customers who buy the most products**.  
- Highlights **customers generating the highest profit**.  
- Useful for loyalty programs and targeted marketing strategies.  

## Features
- Interactive **Page Navigator** for seamless switching between dashboards.  
- Trend analysis (sales, profit, quantity) over time.  
- Modern KPI cards with clean visual layout.  

## Tools & Technologies
- **Power BI Desktop**: Report creation and visualization  
- **DAX**: Measures for dynamic KPIs   


## Insights
- The **Central region** contributes the highest profit ($13.35K).  
- **Technology** is the most profitable category (52%).  
- Five key cities generate the majority of the profit.  
- Top customers drive most of the revenue, highlighting potential targets for loyalty programs.  


## How to Use
1. Open the Power BI file in **Power BI Desktop**.  
2. Use the **Page Navigator** buttons to switch between dashboards.  
3. Apply slicers to filter data by **region**, **category**, or **city**.
